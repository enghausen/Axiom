# Axiom

Axiom is a multi-source addon for Kodi 21/22 with support for custom provider
modules and tight Trakt integration. It allows connecting to multiple
online/offline services at once and viewing their media from a single menu
system. A compatible cloud service is required.

Axiom is a standalone continuation of Seren, based on Prism 4.0.1 and pinned
before Prism's pivot to Simkl and anime of July 2026. Trakt is and remains the
core sync service.

Repository: https://github.com/enghausen/Axiom

## Why Axiom

- Full ownership of the code, no automatic updates from third party repositories
- Trakt stays the sync backend
- Compatible with the Trakt API changes enforced on June 30, 2026 (forced
  pagination and new extended parameter semantics on the watched endpoints)

## Installation

Install from zip in Kodi: `context.axiom` first, then `plugin.video.axiom`.
The addon folder must be the top level entry inside the zip. A repository addon
for automatic updates is planned.
